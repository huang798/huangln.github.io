用Hexo+GitHub架設個人部落格

發表於 2019-05-05

前言

開始學習網頁前端後總覺得資訊量龐大，但一直沒辦法找到一個好地方保存自己的所有筆記。尋覓之下最後找到了Hexo這個部落格框架，只要搭配GitHub就可以輕鬆地建立自己的部落格。

之所以第一篇文就寫Hexo部落格的建立方法是因為自己在建立的過程中一直撞壁，花不少力氣才找到正確的方法，因此就在這裡發文紀錄正確的作法，也希望能給其他找不到方法的人做參考。

Hexo簡介

Hexo是一個快速、簡單且強大的網誌框架。它使用Markdown(或其他渲染引擎)解析文章，並在幾秒鐘內透過漂亮的主題產生靜態檔案。

它的特色包含：

  • 離線也可以進行編輯
  • 架在GitHub上，不需要自己的伺服器
  • 使用 Markdown 語法編輯文章
  • 中文兼容性高
  • 多種不同的網站主題可以選擇
  • 多種插件可以安裝，包含disqus討論功能和瀏覽人數計算等等
